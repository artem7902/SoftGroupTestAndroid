Ссылка на github - https://github.com/artem7902/SoftGroupTestAndroid.
Хранение данный локально - с помощью базы данных SQLite (пакет database).
Пароли хранятся в виде хешей, алгоритм шифрования MD5.
Сессии реализованы с помощью SharedPreferences.
Для валидации email вместо регулярных выражений использовались стандартные функции библиотеки android.
